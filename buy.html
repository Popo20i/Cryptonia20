Guide d'achat

Voici les étapes simples pour acheter le token CT20 sur Uniswap :

  1. ✅ Installez MetaMask (ou un autre portefeuille Ethereum).
  2. 💰 Achetez ou transférez de l’ETH sur votre portefeuille.
  3. 🔁 Allez sur Uniswap.
  4. ➕ Ajoutez le token CT20 avec cette adresse :
    0xFd655eB8fB0296DeeBFdFdc18a05a522186985e1
  5. 🚀 Choisissez la quantité de CT20 et confirmez la transaction.
